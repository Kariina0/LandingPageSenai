# EduTech Future - Landing Page

## Sobre o Projeto
Landing Page desenvolvida para a disciplina de DevOps. O objetivo é criar uma página para uma empresa de Educação, focando em boas práticas de código.

## Como Rodar Localmente
1. Clone o repositório.
2. Abra o arquivo `index.html` no seu navegador.

* CSS3
* JavaScript
